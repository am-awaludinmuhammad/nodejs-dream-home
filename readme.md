project name: Dream Home

framework: express

database: postgresql

scope to this project:

this project has 2 types of users:
- admin
- normal

admin users have access to the backoffice area. normal user only have access to the customer area.

admin users ability:
  - master data
    - categories
    - certificates
    - products
  - customers
  - orders

normal users ability:
  - products
    - list
    - detail
  - order
    - placing order
    - history
  - account
    - register
    - login
    - profile management
    - logout
